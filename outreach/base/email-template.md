# Email Template

This is a template for cold outreach emails. Create one of these in each prospect folder as `email.md`.

**Variables available:**
- `{PROSPECT_NAME}` — Prospect company name
- `{CONTACT_NAME}` — Contact person name
- `{WEBSITE_URL}` — Website URL
- `{PORTFOLIO_URL}` — Your portfolio URL for this prospect
- `{YOUR_NAME}` — Your name
- `{YOUR_EMAIL}` — Your email
- `{YOUR_PHONE}` — Your phone number
